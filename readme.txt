This folder is a R project containing all files related to the Shiny application "Interactive Transformation Models".

To run the app, open interactive-transformation-models.Rproj and click on "Run App"."

- interactive-transformation-models.Rproj is the project container. It contains the environment of the app.
- app.R is the application.
- analysis folder and DVC.tgz allow the loading of one of the datasets used in the app.
- manifest.json stores all dependencies of the app, it is used in the publication of the application with Connect Cloud.
- .gitignore lists file to ignore when commiting to GitHub.
- README.md is the info displayed at the top of the GitHub repository.
